SnowBros Remake
A C++ and SFML replica of the classic arcade game where you freeze enemies with snowballs and kick them to clear screens. Built in 3 weeks as my university OOP project, it features solo and local co-op modes, 10 levels with 2 boss fights, an enemy inheritance hierarchy, an in-game shop, a leaderboard, and a menu with music and user login features.

Game Objective
In Snow Bros, the main goal is to clear each screen of all enemies. You throw snowballs at them until they are completely covered and turn into a giant snowball. Once covered, the snowball can be kicked to roll through platforms, bouncing off walls and defeating any remaining enemies in its path.

Features
Game Modes and Levels
Solo and Local Co-op: Play by yourself or team up with a friend on the same device.
10 Complete Levels: Progress through 10 fully working levels with different layouts, including two dedicated boss fights.

Menu and User Accounts
Main Menu: Includes background music, a New Game option, and a Continue option.
Save and Continue System: The continue feature lets players choose exactly which level to return to, up to the highest level they have completed.
User Accounts: A secure login system for both singleplayer and multiplayer modes that remembers users and passwords.

Gameplay and Economy
Progression: The game difficulty increases as you move through the levels.
In-Game Shop: Earn rewards to purchase powerups that assist you during gameplay.
Leaderboard: A tracking score system linked to a functional leaderboard to display top players.

Technical Design and OOP Structure
Strict Inheritance Hierarchy: Built using an organized base enemy class that different enemy types inherit characteristics and behaviors from.
Object-Oriented Architecture: Leverages core OOP principles across all major systems, including player movement, enemy AI, collision detection, and user data management.

Tech Used
Language: C++
Graphics/Audio Library: SFML (Simple and Fast Multimedia Library)
Core Concepts: Object-Oriented Programming, Game Programming, Memory Management
